### 1.3 섹션별 설계

#### Hero Section
- 서비스 로고/타이틀
- 한 줄 설명
- CTA 버튼 (새 글 작성하기)

#### Features Section
- 4개 핵심 기능 카드
- 아이콘 + 제목 + 간단 설명

#### LLM Providers
- 지원하는 LLM 로고/뱃지

#### Quick Start
- 3단계 시작 가이드

#### Ad Space
- 광고 배너 영역 (placeholder)

#### Support Section
- 후원 링크 (Buy me a coffee, GitHub Sponsors)
- 문의 정보

#### Footer
- 버전 정보
- 크레딧

## 3. 페르소나 시스템

### 3.1 LLM 자동 스타일 분석

사용자가 기존 글을 붙여넣으면 LLM이 글쓰기 스타일을 분석하여 페르소나 설정 자동 생성

#### 분석 프롬프트
```
다음 글의 작성 스타일을 분석해주세요:

[사용자 제공 텍스트]

다음 항목을 JSON으로 출력:
{
  "tone": "professional|casual|friendly|formal",
  "perspective": "first|third",
  "sentenceLength": "short|medium|long",
  "useEmoji": boolean,
  "useHonorific": boolean,
  "vocabulary": "simple|moderate|technical",
  "characteristics": ["특성1", "특성2", "특성3"],
  "commonPhrases": ["자주 사용하는 표현들"],
  "suggestedPromptPrefix": "생성된 프롬프트 프리픽스"
}
```

### 3.2 페르소나 관리 UI

설정 페이지에 "페르소나" 탭 추가

### 3.3 글 작성 시 페르소나 선택

LLM 인디케이터 옆 또는 아래에 페르소나 선택 UI
